## 2026-03-17 21:05 - Build Fix: Playwright Dynamic Import
- Fixed Next.js build failure caused by Turbopack statically analyzing the dynamic import('playwright') in jobsbg.ts
- Changed dynamic import to use Function constructor which hides the import from Next.js static analysis
- Build now passes cleanly with 0 warnings (was failing with "Module not found: Can't resolve 'playwright'")
- All 207 tests still pass
- Root cause: Next.js Turbopack follows all import chains at build time, even dynamic imports wrapped in try/catch
- Fix: Use `new Function('return import("playwright")')()` to completely hide the import from static analysis
